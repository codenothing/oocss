Object Oriented CSS
========================

OOCSS is a new style of writing css with a tree mindset. The parser is PHP based, and is meant to speed up web development by making css easier.


Usage
-----

	$CSSC = new CSSCompressor($css, $options);
	echo $CSSC->css;
